# electron-stock
APP: Inventario local (Electron + SQLite3)

Requisitos:
- Node.js instalado (https://nodejs.org)

Cómo abrir:
1. abrir pasta controle-stock
2. Fazer doble click no arquivo 'run.bat'.
   - Comecara instalar umas dependencias e no final o aplicativo ira abrir.
3. Alternativa: abrir CMD na pasta e rodar:
   npm install
   npm start

Guardar / copiar:
- A base de dados é o arquivo 'inventario.db' na mesma pasta.
